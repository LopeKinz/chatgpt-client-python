

# GPT Client
## A Gui Client made in Python for Chat-GPT

* `Easy Use`
* `Easy Setup`
* `Dark and Light Mode`
* `File Write Mode`
* `Jailbreak Mode`

## Installation

Run with Python

* `Paste OPENAI Api Key into Settings.json`
* `Run the .exe` 
* or

```bash
python3 main.py
```
    
## Full Features list

- Light/dark Mode toggle
- File Write Mode
- Jailbreak Mode
- Coding Mode
- Community Contributions
- Discord Integration
- Clean Simple UI
